# Nike Run Club UI Prototype

This project is a UI/UX concept inspired by the Nike Run Club mobile application.

The prototype was designed in Figma to explore modern mobile fitness interface design, focusing on clean layouts, intuitive navigation, and an engaging user experience for runners tracking their activity.

## Interactive Prototype

[View the Interactive Figma Prototype](https://www.figma.com/design/9pQDTGGAHln2ZNvMdEqJF4/NikeRunClub-UI?node-id=0-1&t=Jaj5vYWxKIWg7P8v-1)

## Project Overview

This prototype explores several key screens commonly found in a fitness tracking application, including:

- Launch screen
- Home dashboard
- Run tracking interface
- Activity history
- Pace metrics
- Club / community section

## Design Tools

- Figma
- Mobile UI design principles
- Interaction prototyping
